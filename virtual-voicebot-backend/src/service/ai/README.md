# ai モジュール

## 目的・責務
- ASR（音声→テキスト）、LLM（テキスト→応答）、TTS（テキスト→音声PCM）の外部サービスクライアントを提供する。
- app からのリクエストに対し、非同期に結果を返す I/F を持つ。

## 詳細設計
- 正本: [docs/ai.md](../../docs/ai.md)

## app から見た I/F
- asr: PCM/WAV 入力 → partial/final テキストを返す。
- llm: ユーザ発話＋履歴 → 応答テキスト/アクションを返す。
- tts: テキスト → PCM チャンク（または WAV）を返す。

注意事項
- セッション制御や通話状態は持たない（app/session に委譲する）。
- 生ネットワーク I/O や SIP/RTP 処理は行わない。
- AI サービスのエンドポイントや認証などはここに閉じ込め、上位には抽象化された結果のみ渡す。
